## 🚀 Contenido del Repositorio
El código está organizado en tres grandes etapas de aprendizaje y aplicación:

## 1. Programación Estructurada
En esta sección se encuentran las bases de mi formación. Se centra en el diseño de algoritmos eficientes, manejo de estructuras de control y tipos de datos.

Conceptos clave: Bucles (for, while), condicionales, recursividad y manejo de arreglos.

Ejemplos: Implementación de algoritmos de ordenamiento y lógica matemática.

## 2. Programación Orientada a Objetos (POO)
Aquí el enfoque cambia hacia la creación de software modular y escalable. He aplicado los pilares fundamentales de la POO para resolver problemas del mundo real.

Abstracción y Encapsulamiento: Diseño de clases con atributos privados y métodos públicos.

Herencia y Polimorfismo: Creación de jerarquías de clases para reutilización de código.

Interfaces: Definición de contratos de comportamiento para desacoplar el código.

## 3. Proyectos en Java
Proyectos integradores que combinan lógica de negocio, persistencia de datos y, en algunos casos, interfaces gráficas. Estos proyectos puede observarlos en la carpeta "Proyectos"

Sistemas de Gestión: Aplicaciones que demuestran el flujo completo de datos.

Algoritmos Avanzados: Implementación de soluciones como el algoritmo de Dijkstra para optimización de rutas.

Herramientas: Uso de Java Swing para interfaces de usuario y manejo de excepciones para robustez.

## 🛠️ Tecnologías y Herramientas
Lenguaje principal: Java (JDK 17+)

Entornos: VS Code

Control de Versiones: Git y GitHub

Sistemas Operativos de desarrollo: Windows y distribuciones Linux (Fedora)
